# Project Title

Playlist Challenge 

# Getting Started

Here we can create new playlist and delete uploaded playlist. We can play and stop the uploaded audio file.


# Installing

## Front End
```
npm Install
```

## Back End
```
npm install
npm install express
npm install multer
npm install cors
```

# Run the application
## Front End 
```
ng serve
```

## Back End

```
node app.js
```
